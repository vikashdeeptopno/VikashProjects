# To check the spending patterns of Males and Females
male_spend = df[df['Gender'] == 'Male']['MonthlySpend']
female_spend = df[df['Gender'] == 'Female']['MonthlySpend']

#Performing t-test
t_stat, p_value = stats.ttest_ind(male_spend, female_spend)
print(f'T Statistic: {t_stat}, P-Value: {p_value}')

# Defining the significance level
alpha = 0.05

# Interpretating the results 
if p_value < alpha:
    print("\nResult: Reject the null hypothesis. The average monthly spend between Males and Females is significantly different.")
else:
    print("\nResult: Fail to reject the null hypothesis. There is no significant difference in average monthly spend between Males and Females.")

T Statistic: 0.3391730320232445, P-Value: 0.7344892727022859

Result: Fail to reject the null hypothesis. There is no significant difference in average monthly spend between Males and Females.

Interpretation:

  • T Statistic is approximately 0.3392 which is close to 0, which means the difference between the average spending of Males and Females are very small.
  • With P-Value being 0.7345, we can say that the average spending pattern of Males and Females are about the same and the small difference we see are likely due to random chance.

# Creating empty list to store spend for each education level
spend_by_edu = []

# To get the list of unique education levels
edu_levels = df['Education'].unique()

# Looping through each education level
for level in edu_levels:
    # Finding all the rows in the DataFrame where the 'Education' column matches the current 'level'
    monthly_spend_level = df[df['Education'] == level]['MonthlySpend']

    # Adding this new data to our list
    spend_by_edu.append(monthly_spend_level)

# Performing ANOVA test to check if the education level impact the monthly average spend
f_stat, p_value = stats.f_oneway(*spend_by_edu)

print(f"F Statistic: {f_stat}, P-Value: {p_value}")

# Defining the significance level
alpha = 0.05

# Interpretating the results 
if p_value < alpha:
    print("\nResult: Reject the null hypothesis. There is a significant difference in the average monthly spend across different education levels.")
else:
    print("\nResult: Fail to reject the null hypothesis. There is no significant difference in average monthly spend across different education levels.")

F Statistic: 0.2288066867370917, P-Value: 0.922359467759936

Result: Fail to reject the null hypothesis. There is no significant difference in average monthly spend across different education levels.

Interpretation:

  • F Statistic is approximately 0.2289 which is ver low and it tells us that the variation in the average spending across the groups is very small.
  • The P-Value is very high (0.9224) which is way beyond the usual 0.05 threshold thus we fail to reject the null hypothesis.
  • This means that there is a 92% chance that the average monthly spend across different education levels are the same.

Conclusion: Every one spends the same across different education levels.
